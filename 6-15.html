LCM OF ANY TWO NUMBER USING HCF

{

j = (n1 for(i=1; i<=j; i++)
{
if(n1%i==0 && n2%i==0)
{
hcf = i;
}
}
/* We know the multiplication of HCF and LCM is equivalant
